### 📚 Formación Académica y Cursos Destacados
Como estudiante de la UNALM, he consolidado una base analítica sólida a través de cursos especializados como:
* **Análisis Multivariado:** Dominio de técnicas como MANOVA, PCA (Análisis de Componentes Principales) y Análisis Factorial para la reducción de dimensionalidad.
* **Estadística No Paramétrica:** Aplicación de pruebas y métricas avanzadas de distancia sin depender de supuestos distribucionales.

* **Bases de Datos y Programación:** Cursos estructurados de Lenguaje de Programación II y Bases de Datos I, enfocados en la arquitectura, prevención de deserción estudiantil y consultas robustas.
